# VideoCharDraw

## 中文

#### 介绍

视频字符画

#### 文件

- CharDraw.py 最初版本字符画
- CharDraw_compress.py 带压缩版本字符画
- CharDraw_thread.py 带压缩和多线程版本字符画

#### 安装教程

1.  直接clone或者下载zip（zip好，多半不会更新了）
2.  下载pillow和opencv-python库（pip国内镜像）

2.  运行
